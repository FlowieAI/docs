---
title: "Get Invoice"
api: "GET /invoices/{id}"
description: "Retrieve a single invoice by ID"
---

## Path Parameters

<ParamField path="id" type="string" required>
  The invoice ID
</ParamField>

## Response

<ResponseField name="id" type="string">
  Unique invoice identifier
</ResponseField>

<ResponseField name="number" type="string">
  Invoice number
</ResponseField>

<ResponseField name="supplier" type="object">
  Supplier information
</ResponseField>

<ResponseField name="amount" type="number">
  Total invoice amount
</ResponseField>

<ResponseField name="currency" type="string">
  Currency code (ISO 4217)
</ResponseField>

<ResponseField name="status" type="string">
  Current invoice status
</ResponseField>

<ResponseField name="line_items" type="array">
  Invoice line items
</ResponseField>

## Example

<CodeGroup>
```bash curl
curl https://back.flowieapp.com/invoices/inv_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
import requests

response = requests.get(
    "https://back.flowieapp.com/invoices/inv_abc123",
    headers={"Authorization": "Bearer YOUR_TOKEN"}
)
invoice = response.json()["data"]
```
</CodeGroup>
