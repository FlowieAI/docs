---
title: "Create Invoice"
api: "POST /invoices"
description: "Create a new invoice"
---

<Info>
  **Coming Soon** - API invoice creation is not yet available. Invoices are currently created through the Flowie web application, file imports, or by uploading invoice documents which are processed via OCR.
</Info>

## Request Body

<ParamField body="number" type="string" required>
  Invoice number
</ParamField>

<ParamField body="supplier_id" type="string" required>
  Supplier ID
</ParamField>

<ParamField body="amount" type="number" required>
  Total invoice amount
</ParamField>

<ParamField body="currency" type="string" required>
  Currency code (ISO 4217)
</ParamField>

<ParamField body="invoice_date" type="string" required>
  Invoice date (YYYY-MM-DD)
</ParamField>

<ParamField body="due_date" type="string" required>
  Payment due date (YYYY-MM-DD)
</ParamField>

<ParamField body="line_items" type="array">
  Invoice line items
</ParamField>

<ParamField body="purchase_order_id" type="string">
  Associated purchase order ID for matching
</ParamField>

## Response

Returns the created invoice object.

## Example

<CodeGroup>
```bash curl
curl -X POST https://back.flowieapp.com/invoices \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "number": "INV-2024-002",
    "supplier_id": "sup_xyz",
    "amount": 2500.00,
    "currency": "EUR",
    "invoice_date": "2024-01-15",
    "due_date": "2024-02-15",
    "line_items": [
      {
        "description": "Software License",
        "quantity": 5,
        "unit_price": 500.00
      }
    ]
  }'
```

```python Python
import requests

response = requests.post(
    "https://back.flowieapp.com/invoices",
    headers={
        "Authorization": "Bearer YOUR_TOKEN",
        "Content-Type": "application/json"
    },
    json={
        "number": "INV-2024-002",
        "supplier_id": "sup_xyz",
        "amount": 2500.00,
        "currency": "EUR",
        "invoice_date": "2024-01-15",
        "due_date": "2024-02-15"
    }
)
invoice = response.json()["data"]
```
