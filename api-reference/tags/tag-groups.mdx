## List Tag Groups

Retrieve all tag groups for an organization.

```bash
curl https://back.flowieapp.com/organizations/org_abc123/tagGroups \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```json
        {
          "tagId": "tag_001",
          "name": "Operations",
```

## Create Tag Group

```
POST /organizations/{org_id}/tagGroups
```

### Request Body

<ParamField body="name" type="string" required>
  Tag group name
</ParamField>

<ParamField body="shortName" type="string">
  Short abbreviation (e.g., "CC" for Cost Center)
</ParamField>

<ParamField body="assignmentRule" type="string" required>
  Assignment rule: `SINGLE`, `MULTIPLE`, `RATIO`, `LEAF_ONLY`
</ParamField>

<ParamField body="description" type="string">
  Description of the tag group
</ParamField>

<ParamField body="objectTypes" type="array">
  Allowed object types for this group
</ParamField>

<ParamField body="businessUnits" type="array">
  Restrict to specific business units (empty = all)
</ParamField>

### Example

```bash
curl -X POST https://back.flowieapp.com/organizations/org_abc123/tagGroups \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Project",
    "shortName": "PRJ",
    "assignmentRule": "MULTIPLE",
    "description": "Project tracking tags",
    "objectTypes": ["invoice", "purchase_order", "purchase_request"]
  }'
```

## Bulk Create Tag Groups

```
POST /organizations/{org_id}/tagGroups/bulk
```

Create multiple tag groups in one request.

```json
{
  "tagGroups": [
    {
      "name": "Cost Center",
      "assignmentRule": "SINGLE"
    },
    {
      "name": "Project",
      "assignmentRule": "MULTIPLE"
    }
  ]
}
```

## Update Tag Group

```
PUT /organizations/{org_id}/tagGroups/{id}
```

## Delete Tag Group

```
DELETE /organizations/{org_id}/tagGroups/{id}
```

<Warning>
  Deleting a tag group removes all tags and object associations within it.
</Warning>

## Update Translations

```
PUT /organizations/{org_id}/tagGroups/{id}/translations
```
