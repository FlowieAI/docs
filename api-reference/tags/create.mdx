---
title: "Create Tag"
api: "POST /tag/tags"
description: "Create a new tag"
---

<Info>
  **Coming Soon** - API tag creation is not yet available. Tags are currently created through the Flowie web application.
</Info>

## Request Body

<ParamField body="name" type="string" required>
  Tag name
</ParamField>

<ParamField body="color" type="string">
  Hex color code
</ParamField>

## Example

```bash
curl -X POST https://back.flowieapp.com/tag/tags \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"name": "Priority", "color": "#FF6600"}'
```

<ResponseExample>
```json
{
  "data": {
    "id": "tag_new",
    "name": "Priority",
    "color": "#FF6600"
  }
}
```
</ResponseExample>
