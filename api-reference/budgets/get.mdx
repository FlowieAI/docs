---
title: "Get Budget"
api: "GET /budgets/{id}"
description: "Retrieve a single budget with details"
---

## Path Parameters

<ParamField path="id" type="string" required>
  Budget ID
</ParamField>

## Example

```bash
curl https://back.flowieapp.com/budgets/bud_123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
