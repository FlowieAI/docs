---
title: "Get Budget"
api: "GET /budget/organizations/{org_id}/budgets/{budget_id}"
description: "Retrieve a single budget with details"
---

## Path Parameters

<ParamField path="org_id" type="string" required>
  Organization ID
</ParamField>

<ParamField path="budget_id" type="string" required>
  Budget ID
</ParamField>

## Example

```bash
curl https://back.flowieapp.com/budget/organizations/org_abc123/budgets/bud_123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
