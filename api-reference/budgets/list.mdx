---
title: "List Budgets"
api: "GET /budget/organizations/{org_id}/budgets"
description: "Retrieve a list of budgets"
---

## Query Parameters

<ParamField query="type" type="string">
  Filter by type: `annual`, `quarterly`, `monthly`, `project`
</ParamField>

<ParamField query="period" type="string">
  Filter by period (YYYY or YYYY-QN or YYYY-MM)
</ParamField>

<ParamField query="closed" type="boolean">
  Filter by closed status
</ParamField>

<ParamField query="businessUnitId" type="string">
  Filter by business unit
</ParamField>

## Example

```bash
curl https://back.flowieapp.com/budget/organizations/org_abc123/budgets \
  -H "Authorization: Bearer YOUR_TOKEN"
```
