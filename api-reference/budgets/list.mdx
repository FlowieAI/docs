---
title: "List Budgets"
api: "GET /budgets"
description: "Retrieve a list of budgets"
---

## Query Parameters

<ParamField query="type" type="string">
  Filter by type: `annual`, `quarterly`, `monthly`, `project`
</ParamField>

<ParamField query="period" type="string">
  Filter by period (YYYY or YYYY-QN or YYYY-MM)
</ParamField>

## Example

```bash
curl https://back.flowieapp.com/budgets \
  -H "Authorization: Bearer YOUR_TOKEN"
```

<ResponseExample>
```json
{
  "data": [
    {
      "id": "bud_123",
      "name": "Marketing Q1 2024",
      "type": "quarterly",
      "allocated": 50000.00,
      "committed": 15000.00,
      "spent": 10000.00,
      "available": 25000.00,
      "currency": "EUR"
    }
  ]
}
```
</ResponseExample>
