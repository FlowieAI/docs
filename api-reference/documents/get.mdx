---
title: "Get Document"
api: "GET /documents/{id}"
description: "Retrieve a document with all versions and metadata"
---

## Path Parameters

<ParamField path="id" type="string" required>
  Document ID
</ParamField>

## Response

Returns the document with:
- Document metadata
- All versions
- OCR data (if processed)
- Access control list
- Linked objects

## Example

```bash
curl https://back.flowieapp.com/documents/doc_xyz789 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
