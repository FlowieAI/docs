---
title: "List Documents"
api: "GET /doc/organizations/{org_id}/documents"
description: "Retrieve documents with filtering and pagination"
---

## Overview

The Document API manages file storage, versioning, and access control. Documents can be linked to any domain object (invoices, purchase orders, contracts, etc.).

## Path Parameters

<ParamField path="org_id" type="string" required>
  Organization ID
</ParamField>

## Query Parameters

<ParamField query="folder_id" type="string">
  Filter by folder
</ParamField>

<ParamField query="object_type" type="string">
  Filter by linked object type: `invoice`, `purchase_order`, `contract`, `supplier`
</ParamField>

<ParamField query="object_id" type="string">
  Filter by linked object ID
</ParamField>

<ParamField query="status" type="string">
  Filter by status: `active`, `archived`, `processing`
</ParamField>

<ParamField query="page" type="integer" default="1">
  Page number
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Items per page (max 100)
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of document objects
</ResponseField>

## Example

```bash
curl "https://back.flowieapp.com/doc/organizations/org_abc123/documents" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
