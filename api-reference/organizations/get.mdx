---
title: "Get Organization"
api: "GET /organizations/{id}"
description: "Retrieve a single organization by ID"
---

## Path Parameters

<ParamField path="id" type="string" required>
  Organization ID
</ParamField>

## Response

Returns the organization object with full details including:
- Organization profile (name, VAT, addresses)
- Business units
- Member count
- Configuration settings

## Example

```bash
curl https://back.flowieapp.com/organizations/org_abc123 \
  -H "Authorization: Bearer YOUR_TOKEN"
```
