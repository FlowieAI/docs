---
title: "List Organizations"
api: "GET /organizations"
description: "Retrieve organizations for the authenticated user"
---

## Overview

Returns a list of organizations that the authenticated user has access to.

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination
</ParamField>

<ParamField query="per_page" type="integer" default="20">
  Number of items per page (max 100)
</ParamField>

<ParamField query="search" type="string">
  Search organizations by name
</ParamField>

## Response

<ResponseField name="data" type="array">
  Array of organization objects
</ResponseField>

## Example

<CodeGroup>
```bash curl
curl https://back.flowieapp.com/organizations \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```python Python
import requests

response = requests.get(
    "https://back.flowieapp.com/organizations",
    headers={"Authorization": "Bearer YOUR_TOKEN"}
)
organizations = response.json()["data"]
```
