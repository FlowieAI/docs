## What is Flowie E-Invoicing?

Flowie is an **AI-powered e-invoicing platform** that processes documents using a multi-model ensemble of GPT-4 Vision, Gemini Pro Vision, and Claude Vision. Upload any invoice format and receive structured, validated data.

## Supported Formats

### Unstructured (AI Extraction)

| Format | Extensions |
|--------|------------|
| PDF | `.pdf` |
| Images | `.png`, `.jpg`, `.jpeg`, `.tiff` |
| Office | `.docx`, `.xlsx` |

### Structured (Direct Parsing)

| Standard | Description |
|----------|-------------|
| **PEPPOL BIS 3.0** | EU e-invoicing network |
| **UBL 2.0/2.4** | Universal Business Language |
| **Factur-X** | French standard (PDF + XML) |
| **SAP IDOC** | INVOIC_02 format |

## Processing Pipeline

```
Upload → Format Detection → OCR/Parse → AI Extraction → Validation → Ready
```

<Steps>
  <Step title="Upload">
    Send document to `/doc/organizations/{org_id}/documents`
  </Step>
  <Step title="Detection">
    System identifies format (PDF, XML, hybrid)
  </Step>
  <Step title="Extraction">
    AI ensemble extracts: vendor, amounts, line items, dates
  </Step>
  <Step title="Validation">
    VAT verification, duplicate check, PO matching
  </Step>
</Steps>

## Quick Start

### 1. Upload a Document

```bash
curl -X POST https://back.flowieapp.com/doc/organizations/{org_id}/documents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@invoice.pdf"
```

**Response:**
```json
{
  "data": {
    "id": "doc_xyz789",
    "status": "processing"
  }
}
```

### 2. Check Processing Status

```bash
curl https://back.flowieapp.com/doc/organizations/{org_id}/documents/{doc_id} \
  -H "Authorization: Bearer YOUR_TOKEN"
```

**Response:**
```json
{
  "data": {
    "id": "doc_xyz789",
    "status": "completed",
    "extractionConfidence": 0.94,
    "extractedData": {
      "documentType": "INVOICE",
      "invoiceNumber": "INV-2024-001",
      "vendor": {
        "name": "Acme Corp",
        "vatNumber": "FR12345678901"
      },
      "amounts": {
        "netAmount": 1000.00,
        "taxAmount": 200.00,
        "grossAmount": 1200.00,
        "currency": "EUR"
      }
    }
  }
}
```

### 3. Access Processed Invoice

```bash
curl https://back.flowieapp.com/invoices/{invoice_id} \
  -H "Authorization: Bearer YOUR_TOKEN"
```

## AI Models

| Provider | Model | Role |
|----------|-------|------|
| **OpenAI** | GPT-4 Vision | Primary extraction |
| **Google** | Gemini Pro Vision | Ensemble |
| **Anthropic** | Claude Vision | Ensemble |

The ensemble approach cross-validates results for higher accuracy.

## Extracted Fields

| Category | Fields |
|----------|--------|
| **Header** | Invoice number, dates, currency |
| **Vendor** | Name, VAT, address, bank details |
| **Amounts** | Net, tax, gross, tax breakdown |
| **Lines** | Description, quantity, unit price, tax |
| **References** | PO number, contract, delivery |

## Confidence Scoring

| Score | Action |
|-------|--------|
| **95%+** | Auto-approve candidate |
| **80-94%** | Review recommended |
| **60-79%** | Manual review |
| **<60%** | Manual entry |

## Webhooks

Get notified when processing completes:

```json
{
  "event": "document.processed",
  "data": {
    "documentId": "doc_xyz789",
    "invoiceId": "inv_abc123",
    "status": "completed",
    "confidence": 0.94
  }
}
```

## Authentication

All requests require Bearer token:

```bash
curl https://back.flowieapp.com/invoices \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

See [Authentication](/authentication) for OAuth 2.0 setup.
