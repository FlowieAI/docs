## AI-Powered E-Invoicing

Upload invoices in **any format** - PDF, images, or electronic standards. Flowie's AI ensemble extracts structured data automatically using GPT-4 Vision, Gemini Pro, and Claude.

## Quick Example

```bash
# Upload an invoice
curl -X POST https://back.flowieapp.com/doc/organizations/{org_id}/documents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@invoice.pdf"
```

**Extracted data:**

```json
{
  "documentType": "INVOICE",
  "confidence": 0.94,
  "vendor": {
    "name": "Acme Corp",
    "vatNumber": "FR12345678901"
  },
  "amounts": {
    "net": 1000.00,
    "tax": 200.00,
    "gross": 1200.00
  },
  "lineItems": [...]
}
```
