## Key Features

<AccordionGroup>
  <Accordion title="Invoice Management" icon="file-invoice">

    - Multi-level approval workflows
    - Duplicate detection
    - Three-way matching with POs
  </Accordion>

  <Accordion title="Purchase Orders" icon="cart-shopping">
    Create and manage purchase orders with full lifecycle tracking.
    - Create POs from approved purchase requests
    - Track order status and deliveries
    - Budget allocation per line item

  <Accordion title="Approval Workflows" icon="check-double">

## Base URL

All API requests should be made to:

```
https://back.flowieapp.com
```

For staging/testing:

```
https://back-staging.flowieapp.com
```

## Authentication

The Flowie API uses OAuth 2.0 Bearer tokens for authentication. Include your token in the `Authorization` header:

```bash
curl https://back.flowieapp.com/invoices \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```
