## Supported Invoice Formats

| Format | Type | Description |
|--------|------|-------------|
| **PDF** | Document | Scanned or digital invoices |
| **Images** | PNG, JPG, TIFF | Photos or scans |
| **UBL** | XML | Universal Business Language |
| **Factur-X** | Hybrid PDF | French e-invoicing standard |
| **ZUGFeRD** | Hybrid PDF | German e-invoicing standard |
| **CII** | XML | Cross Industry Invoice |

## Quick Example

```bash
# Upload an invoice
curl -X POST https://back.flowieapp.com/doc/organizations/{org_id}/documents \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -F "file=@invoice.pdf"

# Response: document processing started
{
  "data": {
    "id": "doc_xyz789",
    "status": "processing"
  }
}
```

## Key Features

<AccordionGroup>
  <Accordion title="Multi-Format Support" icon="files">
    Accept invoices in any format your suppliers send - PDF, images, electronic formats (UBL, Factur-X, ZUGFeRD), or even Excel files.
  </Accordion>

  <Accordion title="AI-Powered Extraction" icon="brain">
    Advanced OCR and machine learning extract:
    - Vendor information
    - Invoice number and dates
    - Amounts and currency
    - Line items with descriptions
    - Tax breakdowns
  </Accordion>

  <Accordion title="Automatic Matching" icon="link">
    Invoices are automatically matched to:
    - Existing suppliers in your database
    - Purchase orders for 3-way matching
    - Contracts and agreements
  </Accordion>

  <Accordion title="Compliance Ready" icon="shield-check">
    Built-in support for e-invoicing regulations:
    - French Factur-X mandate
    - German XRechnung/ZUGFeRD
    - Peppol network
    - Italian SDI
  </Accordion>
</AccordionGroup>
